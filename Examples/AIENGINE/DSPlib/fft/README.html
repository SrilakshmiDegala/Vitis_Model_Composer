ðŸ’¡ Note the usage of the To Fixed Size block to convert a variable size signal to a fixed size signal. This is needed to compare the output of the AIE block to the output of the FFT block from Simulink (which is a fixed size signal). Note this is only valid if the variable size signal is fully packed. Otherwise, the To fixed size block will insert zeros when samples are missing. To make sure this is not the case, you can check the status output of the To fixed Size block.
